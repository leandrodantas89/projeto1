Habilidades primárias aprendidas no bootcamp de html

As principais habilidades aprendidas para todo desenvolvedor ou estudando html falando sobre tags são:

  • Tag p
  • Tag a
  • Tag de listas ordenadas e não ordenadas
  • Tag h
  • Links e referências

Tag P

O elemento HTML p representa um parágrafo. Em mídias visuais, parágrafos são representados como blocos indentados de texto com a primeira letra avançada e separados por linhas em branco. Já em HTML, parágrafos são usados para agrupar conteúdos relacionados de qualquer tipo, como imagens e campos de um formulário.

Tag A

O elemento em HTML (ou elemento âncora), com o atributo href cria-se um hiperligação nas páginas web, arquivos, endereços de emails, ligações na mesma página ou endereços na URL. O conteúdo dentro de cada precisará indicar o destino do link.

Tag Listas ordenadas e não ordenadas

O Elemento HTML (ou Elemento HTML de lista ordenada) representa uma lista de itens ordenados. De forma característica esses itens ordenados em uma lista são mostrados com uma contagem que os precede, que pode ser de qualquer tipo, como numerais, letras, algarismos romanos, ou simples símbolos. Esse modelo numerado não é definido na descrição html da página, mas na folha de estilos CSS associada, pela propriedade list-style-type (en-US).

Exemplo de listas ordenadas

  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4
  5. Item 5

O elemento HTML (ou elemento HTML de Lista desordenada) representa uma lista de itens sem ordem rígida, isto é, uma coleção de itens que não trazem uma ordenação numérica e as suas posições, nessa lista, são irrelevantes. Caracteristicamente, os itens em uma lista desordenada são exibidos com um marcador que pode ter várias formas, como um ponto, um círculo, ou um quadrado. O tipo de marcador não é definido na descrição HTML da página, mas na CSS associada, utilizando a propriedade list-style-type (en-US).

Exemplo de listas não ordenadas

  • Item 1
  • Item 2
  • Item 3
  • Item 4
  • Item 5

Tag H

Os elementos HTML h1 até h6 representam seis níveis de título de seção. h1 é o nível de seção mais alto e h6 é o mais baixo.
